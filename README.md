## Installation

In the console under the file that package.json is:
```
npm i
```
```
npm i express
```
```
npm i uuid
```

## Usage and Test

**After the _npm_ has been installed:** 
```
npm start
```

Or:
```
node server.js
```
Now listening http://localhost:3001
